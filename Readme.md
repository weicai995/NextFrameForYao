# NextFrameworkForYao
Mod前置框架Next降妖散记版
Next Mod是基于BepinEx框架的Mod，为游戏降妖散记提供更简易的修改 (此版本基于觅长生版本的Next Mod框架修改)
github地址： https://github.com/weicai995/NextFrameworkForYao

功能
通过编写json文件与表格，实现在游戏中添加或修改数据
